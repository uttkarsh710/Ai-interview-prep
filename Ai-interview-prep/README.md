# AI Interview Prep

AI powered interview preparation platform.

## Features

- User Login & Signup
- AI Interview Questions
- Voice Interview
- Resume Upload
- Admin Dashboard
- Login History

## Tech Stack

Frontend:
React, CSS

Backend:
Node.js, Express.js

Database:
MongoDB

## Run project

Server:

cd server
node index.js

Client:

cd client
npm start
